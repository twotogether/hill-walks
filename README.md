# Hill Walks Tracker

A collection of hill walks organized by region with an interactive map showing all climbed hills.

## Structure

```
docs/
├── index.md                           # Main page
├── map.md                             # Interactive map (with Leaflet.js)
├── conf.py                            # Sphinx configuration
├── _build_gpx.py                      # Script to copy GPX files during build
├── _static/
│   ├── gpx/                           # Copied GPX files (generated by build)
│   ├── gallery.css                    # Image styling
│   └── ...
├── east-lothian/
│   ├── index.md                       # Region index (lists all hills)
│   ├── images/                        # Photos for East Lothian walks
│   ├── gpx/                           # GPX track files
│   ├── north-berwick-law.md
│   └── eildon-mid-hill.md
└── scottish-borders/
    ├── index.md                       # Region index
    ├── images/                        # Photos for Scottish Borders walks
    ├── gpx/                           # GPX track files
    └── ...
```

## Adding a New Hill

1. **Create a new markdown file** in the region folder:
   ```
   docs/[region-name]/[hill-name].md
   ```

2. **Add metadata table** at the top:
   ```markdown
   # Hill Name

   ## Details

   | Field | Value |
   |-------|-------|
   | **Coordinates** | 56.0588, -2.1384 |
   | **Elevation** | 187 m |
   | **Summit feature** | Trig point, view indicator |
   | **Class list** | Marilyn, Hump |
   | **Date of ascent** | DD-MMM-YYYY |

   ## Description
   ...
   ```

3. **Add photos using figure directive** (captions display properly):
   ```markdown
   ## Photos

   ```{figure} images/[region-name]/[image-name].jpg
   Photo caption text
   ```
   ```

4. **Add GPX track file** (optional):
   - Place track file in `docs/[region-name]/gpx/[hill-name].gpx`
   - Build process automatically copies to `_static/gpx/` for the map

5. **Update region index** in `docs/[region-name]/index.md`:
   ```markdown
   ```{toctree}
   :hidden:

   [hill-name]
   ```
   ```

6. **Add photos** to the region's images folder:
   ```
   docs/[region-name]/images/[image-name].jpg
   ```

## Building Locally

```bash
pip install -r requirements.txt
sphinx-build -b html docs _build/html
```

Then open `file:///path/to/hill-walks/_build/html/index.html` in your browser.

Or use a local HTTP server to avoid CORS issues:
```bash
cd _build/html
python -m http.server 8000
# Visit http://localhost:8000
```

## Deployment

Push to `main` branch — GitHub Actions automatically builds and deploys to GitHub Pages at `https://twotogether.github.io/hill-walks/`.

## Features

- **Interactive map**: Leaflet.js map showing all climbed hills as green markers
- **GPX visualization**: Toggle to show/hide GPX trails on the map
- **Responsive images**: Auto-sized photos with captions
- **Region organization**: Hills grouped by region with separate index pages
- **Automatic builds**: GitHub Actions builds and deploys on every push to main

## Customization

Theme settings are in `docs/conf.py`. Edit `html_theme_options` to customize the Furo theme.
